#Kidney-Disease-Prediction

## Overview
This webapp is used for Chronic Kidney Disease Prediction and recommends it, if the patient NEEDS a doctor or NOT. 



## TechStack

A list of technologies used within the project:
* Flask
* Python
* ML Models
* HTML
* CSS
* Heruko
* Numpy/Pandas
* Sklearn
